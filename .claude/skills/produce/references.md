# References — the genre, shot by shot

You cannot watch video. These are written reconstructions of the three
dominant archetypes in modern product-launch films (Linear, Vercel, Raycast,
Arc, Apple-style), distilled from the genre's conventions — read them as shot
lists to steal structure from, not transcripts. After each produced video,
append its annotated filmstrip as a first-party exemplar.

## Archetype 1 — "Keyboard-first dev tool" (Linear/Raycast school)

~35s · dark · 120–128bpm electronic, dry percussive · Arc A (demo-first)

| t | shot | craft notes |
|---|---|---|
| 0.0–1.5 | Tight crop: command palette already open over blurred app, query half-typed, results reordering live | Mid-action hook; cursor blink sells "live"; bg at 4px blur |
| 1.5–3.5 | Enter pressed → action executes → UI responds with one clean state change | Causality chain: keypress flash → 4f → result; thock SFX |
| 3.5–6 | Type card: "Built for speed" → morphs to product wordmark | Reveal on first drop; word swap via masked slide, not fade |
| 6–24 | Proof montage: 4 beats × ~4.5s. Each: 3-word claim lower-third + one interaction filmed tight (create → assign → automate → integrate). Camera pushes in ~2% per beat | Claims stagger in; each beat cut ON downbeat; energy rises by shortening holds 14f→10f |
| 24–28 | Climax: pull back from tight crop to full board as dozens of items cascade-update (signature move at full size) | The one outsized motion; riser into it, impact + duck on land |
| 28–32 | All UI fades to bg luminance; wordmark + "Try it today" + URL | Music resolves; 100f hold |
| 32–35 | Final frame = wordmark on near-black | Loop-rhymes with the dark hook frame |

Steal: causality timing, accelerating montage, ONE pullback climax.
Avoid: their dark-bg-purple-glow look is the most cloned in tech — if you use
this structure, the surface must come from YOUR identity derivation.

## Archetype 2 — "The feature film" (Apple school)

~45s · light or tonal bg · 85–95bpm, warm, spacious · Arc C/E hybrid

| t | shot | craft notes |
|---|---|---|
| 0–3 | Huge serif/display claim fills 70% of frame, enters word-by-word with blur-settle | Type IS the imagery; 2 words visible by frame 8 |
| 3–8 | Claim lines swap (mask-up) while bg gradient slowly breathes | Each line ≤5 words; holds follow reading-time rule exactly |
| 8–30 | Alternation: type statement → floating product detail (device/UI fragment in 3D-ish space, parallax drift) → type → detail. 5–7 alternations, each 3–4s | The alternation IS the rhythm signature; details never show whole UI, only the beautiful part |
| 30–38 | Details converge: fragments fly to assemble the full product view | Convergence = climax; widest shot of the video |
| 38–45 | Product name set small and confident, centered; one-line promise; URL | Confidence = small type + lots of air + long hold |

Steal: type-led storytelling, alternation rhythm, convergence climax,
reading-time discipline.
Avoid: without truly beautiful product fragments and disciplined copy this
collapses into a slideshow; it's the hardest archetype to earn.

## Archetype 3 — "Infra/terminal launch" (Vercel school)

~30s · near-black, one electric accent · 110–120bpm minimal techno · Arc B or D

| t | shot | craft notes |
|---|---|---|
| 0–2.5 | Terminal, real prompt; command types on (real cadence: bursts + pauses) and hits Enter | String-slicing typewriter; cursor block; muted viewer reads the command = the hook |
| 2.5–5 | Output streams; one line matters — camera pushes onto it, rest dims to 40% | Selective focus = narration without VO |
| 5–8 | The line's METRIC jumps out of the terminal, becomes 140px display type counting up | Signature move: data escaping the terminal; tabular-nums; tick SFX |
| 8–20 | 3 proof beats: diagram nodes lighting in sequence / dashboard panel / second terminal interaction. Accent color = "alive" things only | Diagrams build edge-by-edge (paths drawing), never appear whole |
| 20–25 | Climax: zoom out — the one node becomes a global mesh propagating (signature move, full size) | Scale jump small→planetary is the emotional payload |
| 25–30 | Wordmark + `npm i <thing>` in a styled code chip + URL | The install command IS the CTA for devs; 90f+ hold |

Steal: terminal realism (timing!), data-becomes-display-type, accent as
semantic "live" marker, install-command CTA.
Avoid: don't fake terminal output aesthetics (wrong prompt, fake commands) —
this audience notices in one frame.

## Cross-cutting genre rules

- Every archetype: ONE climax, placed at 70–85% of runtime, on the music's
  biggest moment.
- Logos appear at reveal and CTA, nowhere else. Nobody falls in love with a
  logo.
- The grade (grain/vignette/glow) unifies composited elements — UI screenshots
  dropped raw onto a styled bg always look pasted until graded.
- When in doubt between two shots, choose the one with fewer elements.

## X-viral launch patterns (from analysis of high-engagement launch videos)

Distilled from publicly-posted, high-engagement product launches on X
(contrarian-claim hooks + autonomous-agent demos, live real-time tool demos,
agent/assistant showcases, and public "best launch video" breakdowns):

**Hook tactics that stop the scroll (first 1-3s must be unmissable, work muted):**
- Provocative contrast claim + instant visual "magic" payoff. Bold "world's first", contrarian ("AI is making X worse" vs solution), or "AI now does Y faster/better than human". Cut immediately to live demo of the product doing impressive things (agents working autonomously in parallel; live tutor pointing/sketching on screen like human next to you; multi-monitor activation on clap).
- Mid-action multi-layer demo: Product already "on", with constant background elements + micro-updates (data streams, tool panels ticking, text resolving/popping, parallel processes). Frame must feel alive — no static, things happening everywhere to fight swipe.
- Data/stats pain + re-hook every 5-8s with new layer/action/angle. Pattern interrupt or number. No logo, no "introducing", no slow fades. Works silent first (big text + visuals tell story); audio rewards (distinctive tap/whoosh on first magic moment).

**Pacing, visuals & production for retention (fast, professional, engaging on X):**
- "Orchestrated velocity" or "relentless but clean": Fast cuts, overlapping parallel actions (multiple agents/tools running in background while main story lands), constant micro-motion (streaming text, resolves with pops/ripples, background layers always updating with parallax). No long static holds (1-2s max). Accelerating energy toward climax.
- High-fidelity live demo of the "impossible" (real-time UI updating, not pre-captured stills). Layered depth (dominant focal + active background layers). Professional native chrome + clean, high-contrast typography. Dynamic range: whips, overlaps, glows on key resolves.
- Thread-optimized: 30-60s hero video + short 10-70s clips (one hook/pain/demo/cred per clip) for algorithm reach.

**Narrative that drives shares:**
- Hook (contrast claim + live demo) → Specific problem with relatable stats/data → Live solution (parallel "magic" with background activity) → Cred/proof (team background, real usage) → Generous low-friction CTA (free trial, "comment for invite", link).
- Emotional: Bad current (frustration, bots/cheating/manual toil) → empowered future ("this just works locally/respectfully"). "Changes everything" without hype.

**What kills virality:** Static screen recordings, feature-list montages without live parallel action, slow builds, no background dynamism, weak first 3s (logo or talk), flat compositions.

Steal for our system (see direction.md + craft.md): force the hook to a
contrarian claim + instant multi-layer live demo with constant background
elements. Default rhythm "orchestrated velocity with constant micro-updates".
Signature moves should enable parallel live streams/resolves/handoffs.
